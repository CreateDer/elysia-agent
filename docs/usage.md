Slash Commands - Direct invocation of tools and workflows

Command list
/elysia-expert:issue [arguments]
Arguments is github issue no that contain the problem described in the issue

/elysia-expert:task [arguments]
Arguments is task file name in /tasks/[arguments].md no that contain the problem described in the task

/elysia-expert:prompt [arguments]
Arguments is prompt statement  that contain the problem described what you want elysia exprt agent to do for ex. "Implement apis for game item store [more specification information.] and provide swagger apis document"
